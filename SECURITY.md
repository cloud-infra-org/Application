# Security Policy

Application security is important to us, and this document explains how to report security concerns safely.

## Supported Versions
Only the latest released version of the application is supported.
Security fixes are applied to supported versions only.

## Reporting a Vulnerability
If you find a security issue, please do not report it publicly.
Create a private security report or contact the maintainers directly.
Provide clear steps to reproduce the issue and its possible impact.

## What to Include
- A detailed description of the vulnerability
- Steps to reproduce the issue
- Any logs, screenshots, or examples if available

## Response Process
The team will review the report as soon as possible.
Valid issues will be fixed and released in a timely manner.
You will be notified once the issue is resolved.

Thank you for helping keep this application secure.
